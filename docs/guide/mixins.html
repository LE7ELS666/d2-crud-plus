# 如何工作

d2CrudPlus是通过vue的混入功能开始工作的(类似类的继承)
混入之后，该页面将继承d2CrudPlus.crud的方法，帮助您简化d2-crud-x和d2-crud的使用

<template>
...
</template>
<script>
import { d2CrudPlus } from 'd2-crud-plus'
export default {
  mixins: [d2CrudPlus.crud], // 最核心部分，继承d2CrudPlus.crud
}
</script>

# 初始化过程

  1. created()中开始crud初始化
  2. 页面配置crudOptions覆盖全局配置commonOption
  3. 遍历crudOptions.columns，详细过程
  4. －－根据type获取默认字段配置
  5. －－使用用户的配置覆盖默认配置
  6. －－search中的配置覆盖form的配置生成字段查询配置
  7. －－addForm中的配置覆盖form的配置生成字段添加配置
  8. －－editForm中的配置覆盖form的配置生成字段编辑配置
  9. －－view中的配置覆盖form的配置生成字段查看配置（仅当componentType='form'）
  10. 处理分组配置
  11. 生成最终的crud配置，输出日志 crud init {crud}（你可以在此检查生成的配置是否正确）
  12. 将crud配置传入d2-crud-x组件
  13. 触发doSearch方法执行pageRequest获取数据

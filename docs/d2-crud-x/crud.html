  • 说明: 显示的数据
  • 类型: Array
  • 可选值: 无
  • 默认值: 无

# columns

  • 说明: 表头数据，具体配置项请参考 columns
  • 类型: Array
  • 可选值: 无
  • 默认值: 无

# options

  • 说明: 表格配置数据，具体配置项请参考 options
  • 类型: Object
  • 可选值: 无
  • 默认值: 无

# loading

  • 说明: 表格加载状态
  • 类型: Boolean
  • 可选值: 无
  • 默认值: false

# loading-options

  • 说明: 表格加载配置数据，具体配置请参考 loading-options
  • 类型: Object
  • 可选值: 无
  • 默认值: 无

# index-row

  • 说明: 索引，具体配置项请参考 index-row
  • 类型: Object / Boolean
  • 可选值: 无
  • 默认值: false

# selection-row

  • 说明: 多选，具体配置项请参考 selection-row
  • 类型: Object / Boolean
  • 可选值: 无
  • 默认值: false

# expand-row

  • 说明: 行展开，具体配置项请参考 expand-row
  • 类型: Object / Boolean
  • 可选值: 无
  • 默认值: false

# add-title

  • 说明: 新增表单标题

  • 可选值: 无
  • 默认值: 无

# add-template

  • 说明: 新增表单数据模板，具体配置请参考 add-template
  • 类型: Object
  • 可选值: 无
  • 默认值: 无

  • 说明: 编辑表单标题
  • 类型: String
  • 可选值: 无
  • 默认值: 无

# edit-template

  • 说明: 编辑表单数据模板，具体配置请参考 edit-template
  • 类型: Object
  • 可选值: 无
  • 默认值: 无

# view-title

  • 说明: 查看表单标题

  • 默认值: 无

# view-template

  • 说明: 查看表单数据模板，具体配置请参考 view-template

  • 可选值: 无
  • 默认值: 无

  • 说明: 表单配置数据，具体配置请参考 form-options

  • 可选值: 无
  • 默认值: 无
